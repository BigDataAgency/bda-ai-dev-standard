- Claude Code interactive: use `/weekly-focus` after installing `claude/commands/weekly-focus.md` into `.claude/commands/`.

- Claude coworker: paste/reference `claude-coworker/prompts/weekly-focus.md`.
- Codex: reference this command path from `AGENTS.md` instructions.
